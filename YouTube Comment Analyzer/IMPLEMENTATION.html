YouTube Comment Analyzer

Here's a quick and easy way to do it:

Follow this steps

Step 1:Choose the video you want to analyze

Step 2:Copy URL/Link of the video.

Step 3: Extract the Video ID: The YouTube video URL typically contains a unique video ID at the end of the URL. The video ID is a string of characters that uniquely identifies the video. It usually follows the 'v'= parameter in the URL. For example, in the URL https://www.youtube.com/watch?v=dQw4w9WgXcQ, the video ID is dQw4w9WgXcQ.

the Video ID: Locate the video ID in the URL and copy it. You can do this by selecting the video ID portion of the URL and then right-clicking and choosing "Copy," or by using the keyboard shortcuts mentioned earlier (Ctrl + C or Cmd + C).

Step 4:Paste Url/Link into input box.

Step 5:Click on Analyze
